# 🛍️ eShop

This project serves as a **learning resource** for anyone interested in exploring what (in my opinion) is a **good solution with clean architecture**.  
It’s still **in progress**, but the goal is to provide a solid reference implementation that demonstrates best practices for building a modern, maintainable application.  

---

## 🚀 Goals
- Showcase **clean architecture** principles.  
- Provide a **practical learning project** for developers.  
- Highlight **good coding practices**, maintainability, and scalability.  
- Serve as a reference for building real-world e-commerce systems.  

---

## 🛠️ Tech Stack (Planned / Current)
- **Backend**: .NET / C#  
- **Frontend**: Angular (planned)  
- **Database**: PostgreSQL  
- **Infrastructure**: Azure Cloud Services, CI/CD Pipelines, Docker (TBD)
